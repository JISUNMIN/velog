`Array.prototype.findIndex()`는 배열에서 **조건을 만족하는 첫 번째 요소의 인덱스(index)** 를 반환합니다.  
만약 조건을 만족하는 요소가 없다면 **`-1`을 반환**합니다.

---

## 🧩 기본 문법
>```js 
array.findIndex(callback(element, index, array), thisArg?)

1. element - 현재 검사 중인 배열의 요소 (예: arr[0],arr[1]...)
2. index? - 현재 요소의 인덱스
3. array? - 호출한 원본 배열 자체
```

## 🛠️ 기본 예시

const arr = [10, 20, 30];

arr.findIndex((n) => n === 20); // 👉 1 (조건을 만족하는 첫 번째 요소)
arr.findIndex((n) => n === 99); // 👉 -1 (조건을 만족하는 요소 없음)

## 💡 사용 예시

React 등에서 특정 데이터를 리스트에서 찾을 때 유용하게 쓰입니다.

```ts
const selectedIndex = filteredProjects.findIndex(
  (project) => String(project.id) === selectedProjectId
);
```
filteredProjects 배열 안에서 project.id가 URL 쿼리의 selectedProjectId와 같은지 비교하고
처음 조건을 만족하는 프로젝트의 인덱스를 반환 selectedIndex가 -1이면 해당하는 프로젝트가 없는 것입니다.

## ⚠️ 주의할 점
반환값은 인덱스입니다. 요소 자체가 아니라는 점에 주의하세요.

조건을 만족하는 요소가 없으면 -1이므로, 후속 로직에서 반드시 체크해야 합니다.

```ts
if (selectedIndex !== -1) {
  // 해당 요소 존재
  const element = itemRefs.current[selectedIndex];
  element?.focus();
}
```

findIndex()는 조건을 만족하는 첫 번째 요소의 인덱스만 반환합니다.
즉, 배열에서 여러 요소가 조건에 맞더라도 오직 **첫 번째**로 조건을 만족하는 요소 하나만 찾고 멈춥니다.

```ts

const arr = [5, 12, 8, 130, 44];

// 10보다 큰 첫 번째 요소의 인덱스를 찾음
const index = arr.findIndex(num => num > 10);

console.log(index); // 1 (값 12의 인덱스)
```

## ❓ 조건을 만족하는 모든 요소의 인덱스를 찾고 싶다면?

조건을 만족하는  **모든 요소의 인덱스가 필요하다면 직접 구현해야 합니다.**
[reduce](https://velog.io/@sunmins/4%EC%9E%A5-%EB%B0%B0%EC%97%B4-%ED%95%A8%EC%88%98-reduce-%EB%88%84%EC%A0%81-%EB%B3%80%ED%99%98-%EC%9A%94%EC%95%BD%EC%9D%98-%EB%81%9D%ED%8C%90%EC%99%95)나 [map](https://velog.io/@sunmins/2%EC%9E%A5-map-vs-forEach-%EB%B9%84%EA%B5%90-%EB%B0%98%ED%99%98%EA%B0%92-%EC%9C%A0%EB%AC%B4%EC%99%80-%EB%AA%A9%EC%A0%81-%EC%B0%A8%EC%9D%B4)과 [filter](https://velog.io/@sunmins/1%EC%9E%A5-Array-vs-Object-%ED%95%A8%EC%88%98-%EB%B9%84%EA%B5%90-%EA%B0%92-%EC%B0%BE%EA%B8%B0%EC%99%80-%ED%95%84%ED%84%B0%EB%A7%81-%ED%95%B5%EC%8B%AC-%ED%95%A8%EC%88%98find-filter-Object.keys-%EB%93%B1)등 으로 직접 구현합니다.

```ts
1. reduce 

const arr = [5, 12, 8, 130, 44];

const allIndexes = arr.reduce((acc, num, idx) => {
  if (num > 10) acc.push(idx);
  return acc;
}, []);

console.log(allIndexes); // [1, 3, 4]

2. map filter

const allIndexes = arr
  .map((item, idx) => (item > 10 ? idx : -1))  // 조건에 맞으면 인덱스, 아니면 -1
  .filter(idx => idx !== -1);                   // -1 제거

console.log(allIndexes); // [1, 3, 4]

```

## ❓ 객체에서 쓰고 싶다면?
Object는 배열이 아니기 때문에 .findIndex() 같은 배열 메서드는 쓸 수 없지만,
[Object.entries](https://velog.io/@sunmins/3%EC%9E%A5-%EA%B0%9D%EC%B2%B4%EC%97%90%EC%84%9C-%EB%B0%98%EB%B3%B5%EB%AC%B8%EC%9D%80-%EC%96%B4%EB%96%BB%EA%B2%8C-%EC%8D%A8%EC%95%BC-%ED%95%A0%EA%B9%8C-Object.entries.map-%ED%99%9C%EC%9A%A9%EB%B2%95)(), Object.keys(), Object.values() 등을 사용해서 배열로 바꾸면 사용할 수 있습니다.

```ts
const obj = { a: 10, b: 20, c: 30 };


const entries = Object.entries(obj)
// [['a', 10], ['b', 20], ['c', 30]];

// 구조 분해 없이
entries.findIndex((element) => element[1] > 10);

// 구조 분해 사용
entries.findIndex(([key, value]) => value > 10);

```

## ✅ 요약 정리

- 배열에서 **조건을 만족하는 첫 번째 요소의 인덱스**를 반환
- 조건을 만족하는 요소가 없으면 **-1 반환**
- 요소 자체가 아닌 **인덱스**를 반환하므로 주의
- React 등에서 특정 항목의 위치를 찾을 때 유용

### 모든 조건에 맞는 인덱스가 필요할 때
- `reduce`, `map` + `filter` 조합으로 직접 구현

### 객체에서 사용하려면?
- `Object.entries()` 등을 사용해 배열로 변환 후 사용
