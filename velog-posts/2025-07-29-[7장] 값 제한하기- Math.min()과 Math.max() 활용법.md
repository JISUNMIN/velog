 이번 글에서는 숫자를 다룰 때 자주 사용하는 `Math.min()`과 `Math.max()` 함수에 대해 알아보고, 실제 코드에서 어떻게 활용되는지 살펴보겠습니다.

---

## 🔢 Math.min()과 Math.max()란?

- **Math.min()** : 여러 숫자 중 가장 작은 값을 반환합니다.  
- **Math.max()** : 여러 숫자 중 가장 큰 값을 반환합니다.

```js
Math.min(5, 10, 3); // 3 반환
Math.max(5, 10, 3); // 10 반환
```

---

### ⚠️주의: 배열을 바로 인자로 넘기면 안됩니다.

```js
const arr = [5, 10, 3];
Math.min(arr); // NaN 반환
Math.max(arr); // NaN 반환
```

`Math.min()`과 `Math.max()`는 여러 숫자를 각각의 인자로 받기 때문에, 배열을 하나의 인자로 넘기면 함수가 제대로 작동하지 않습니다.

---

### ✅ 배열 사용시 전개 연산자(...)를 써야합니다. 

전개 연산자는 배열을 개별 인자들로 펼쳐주는 역할을 합니다.

```js
const arr = [5, 10, 3];
Math.min(...arr); // 3 반환
Math.max(...arr); // 10 반환
```

즉, `Math.min(...arr)`는 내부적으로 `Math.min(5, 10, 3)`과 동일하게 동작합니다.

### ❓ 전개 연산자가 꼭 필요한 이유 

`Math.min()`과 `Math.max()` 함수는 여러 개의 숫자를 **각각의 인자**로 받도록 설계되어 있습니다.

예를 들어:

```js
Math.min(400, 500, 600); // 400 반환
```

하지만 배열을 바로 넘기면 아래처럼 동작합니다:

```js
const arr = [400, 500, 600];
Math.min(arr); // NaN 반환
```

이유는, 배열 `arr` 자체가 하나의 인자로 전달되기 때문입니다. 숫자들이 분리되어 인자로 들어가는 것이 아닙니다.

따라서 전개 연산자(`...`)를 사용해서 배열을 펼쳐야 합니다.

```js
Math.min(...arr); // 400 반환
```

## 두 함수를 함께 쓰는 경우 — 값 제한하기 

두 함수를 함께 쓰는 이유는 어떤 값이 최소값과 최대값 사이에 있도록 제한(clamp)하기 위해서입니다.

```js
const clampedValue = Math.min(Math.max(value, min), max);
```

- 먼저 `Math.max(value, min)`으로 `value`가 `min`보다 작으면 `min`을 반환해 값을 최소값 이상으로 맞춥니다.  
- 그다음 `Math.min(..., max)`으로 최대값을 제한합니다.

---

## 실제 예시: 가로 패널 크기 제한하기 📐

아래 코드는 사용자가 마우스로 드래그해서 패널 크기를 조절할 때, 최소 400px, 최대 800px 사이로 크기를 제한하는 로직입니다.

```ts
const handleMouseMove = (e: MouseEvent) => {
  if (!resizing.current) return;

  // 화면 오른쪽 끝에서 마우스 X 좌표를 빼서 너비 계산
  const newWidth = window.innerWidth - e.clientX;

  // newWidth가 400보다 작으면 400, 800보다 크면 800으로 제한
  setPanelWidth(Math.min(Math.max(newWidth, 400), 800));
};
```

- `window.innerWidth - e.clientX`는 현재 마우스 위치를 기준으로 오른쪽 패널의 너비를 구하는 공식입니다.  
- `Math.max(newWidth, 400)`은 패널 너비가 최소 400px 이상이도록 보장합니다.  
- `Math.min(..., 800)`은 최대 너비를 800px로 제한합니다.

---

## 정리 📝

- `Math.min()`과 `Math.max()`는 숫자 중 최소/최대 값을 쉽게 찾을 수 있는 함수입니다.  
- 두 함수를 조합해 값을 특정 범위 내로 제한할 수 있습니다.  
- UI 개발 시, 크기 조절 등 값의 범위를 제한할 때 자주 활용됩니다.  
- 배열을 인자로 바로 넘기지 말고 전개 연산자(...)를 사용해야 함수가 제대로 동작합니다.
